# ph-econ-index-mds
This study applies Multidimensional Scaling (MDS) to analyze and construct a composite statistical index using major economic indicators in the Philippines, providing insights into the interrelationships and comparative structure of national economic performance metrics.

---

# A Multidimensional Scaling Analysis of Statistical Index Construction using Major Economic Indicators in the Philippines

*Author:* Celmar Bacasma  
*Affiliation:* University of Science and Technology of Southern Philippines  
*Date:* 2025-11-12

---

## Description
This repository contains the full research project titled:

*"A Multidimensional Scaling Analysis of Statistical Index Construction using Major Economic Indicators in the Philippines"*

It includes the cover, chapters, methodology, results, references, and appendices.  
The project analyzes the construction of statistical indices using multidimensional scaling and major economic indicators in the Philippines.

1. Start with 01_cover_and_front_matter.pdf for the abstract and front matter.
